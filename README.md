# Varun Junnuthula — Portfolio

Personal portfolio website.

**Live:** https://varun-junnuthula.vercel.app

A single, dependency-free `index.html` (HTML / CSS / vanilla JS) — no build step.
The contact form delivers messages via [Web3Forms](https://web3forms.com), and the
résumé is available at `/resume.pdf`.

## Editing
- **Projects:** edit the `PROJECTS` array near the bottom of `index.html`.
- **Experience / text:** plain HTML in `index.html`.
- **Résumé:** replace `resume.pdf`.

## Deployment
Hosted on [Vercel](https://vercel.com). Every push to `main` publishes automatically.
